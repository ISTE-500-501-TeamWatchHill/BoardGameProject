# BoardGameProject

### Set up
##### Clone Repo
```
git clone https://github.com/ISTE-500-501-TeamWatchHill/backend.git && cd ./backend
```

##### Set up config and credentials
```
cp ./.env.example ./.env
```

##### Run app
```
npm start index.js
```

##### How to run API tests
```
npm run test
```

##### How to run API tests
```
follow instructions in __tests__/api/api_spec.js
```
